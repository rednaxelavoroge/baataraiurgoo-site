Ключ доступа для публикации

Чтобы правки уходили на живой сайт, нужен персональный ключ GitHub. Вводится один раз и хранится только в этом браузере — на другом компьютере или в другом браузере его нужно будет ввести заново. У ключа есть срок действия (вы задаёте его при создании) — когда он истечёт, нужно будет создать новый и вставить сюда.

  1. Откройте создание токена (Fine-grained).
  2. Repository access → Only select repositories → baataraiurgoo-site.
  3. Permissions → Contents → Read and write.
